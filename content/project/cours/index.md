**Курс "Системный администратор Linux с нуля"** - это программа обучения от Selectel, направленная на освоение востребованной IT-профессии с полного нуля.

### О проекте

Курс охватывает все ключевые аспекты работы системного администратора Linux:
- Основы работы с командной строкой Bash
- Управление пользователями и правами доступа
- Базовые команды Linux
- Управление файлами и каталогами
- Использование справочных систем, работа с текстовыми файлами и логами
- Управление процессами
- Настройка сети и SSH
- Управление пакетами
- Управление логами

### Что я освоил

В рамках курса я приобрел практические навыки:
- **Установка и настройка** серверов 
- **Настройка сетевых служб** (SSH)
- **Мониторинг систем** с помощью htop, zabbix
- **Создание bash-скриптов** для автоматизации рутинных задач
- **Работа с системами контроля версий** Git
- **Управление доступом** с помощью специальных разрешений

- Автоматизировала процессы 
- Написала серию скриптов для автоматизации административных задач
- Освоила диагностику системных проблем
- Получила фундаментальные знания для работы Linux-администратором
